# Day 12 - Your Task

## Toolbar with responsive collapsing slots

- Build a Toolbar component with three named slots: start, center, end.
- On smaller screens, center content should move into an overflow menu (implement simple collapse behavior).
- Deliverable: Toolbar.jsx, demo page showing responsiveness and slot usage.

Learning: slot handling and responsive composition.

## Toast/Notification System with slot-based template

- Implement a Toast system where each toast accepts slots: icon, message, action.
- Provide an API to register toast templates and show toasts with different slot content.
- Deliverable: toast manager , example templates (success, error, info).

Learning: dynamic slot templates and programmatic rendering.
